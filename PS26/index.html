PICT B.E. Course Structure

First Year (FE)

  • Engineering Mathematics-I: Fundamentals of calculus, differential equations.
  • Engineering Physics: Mechanics, waves, optics, and quantum theory basics.
  • Engineering Chemistry: Organic/inorganic chemistry, environmental science.
  • Basic Electrical Engineering: Circuit analysis, AC/DC fundamentals.
  • Engineering Graphics: Orthographic projection, isometric views.
  • Basic Workshop Practice: Hands-on experience with fitting, carpentry, welding.

Second Year (SE)

  • Data Structures: Arrays, linked lists, trees, and graphs.
  • Computer Organization: CPU architecture, memory systems, ALU design.
  • Discrete Mathematics: Set theory, logic, combinatorics, graph theory.
  • Object-Oriented Programming: Classes, inheritance, polymorphism (C++/Java).
  • Digital Logic Design: Boolean algebra, flip-flops, combinational circuits.

Third Year (TE)

  • Database Management Systems: SQL, normalization, transactions.
  • Operating Systems: Process management, memory, file systems.
  • Computer Networks: TCP/IP, OSI model, routing algorithms.
  • Software Engineering: SDLC, UML, Agile methodologies.
  • Theory of Computation: Automata, Turing machines, complexity.

Final Year (BE)

  • Machine Learning: Supervised, unsupervised learning, regression, classification.
  • Cloud Computing: Virtualization, cloud models, services (IaaS, PaaS, SaaS).
  • Information and Cyber Security: Cryptography, threats, firewalls.
  • Electives: Advanced topics like Blockchain, NLP, IoT, etc.
  • Project Work: Capstone project development and presentation.
